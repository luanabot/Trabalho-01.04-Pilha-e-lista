# Sistema de Pilhas e Filas (Java)

## 📌 Descrição

Projeto desenvolvido em Java com o objetivo de demonstrar o funcionamento das estruturas de dados **Pilha** e **Fila**.

O sistema possui dois módulos principais:

- 🌐 **Navegador (Pilhas):** simula as ações de voltar e avançar páginas de um navegador web.
- 🏥 **Hospital (Fila):** simula o atendimento de pacientes por ordem de chegada.

---

## 🛠️ Tecnologias Utilizadas

- Java
- IntelliJ IDEA
- Biblioteca `java.util`

---

## ▶️ Como Executar no IntelliJ IDEA

1. Abra o **IntelliJ IDEA**.
2. Clique em **Open** e selecione a pasta do projeto.
3. Abra o arquivo:

4. Clique no botão ▶️ ao lado do método `main`.
5. O sistema será executado no console.

---

## 🖥️ Funcionamento do Sistema

Ao iniciar o programa, será exibido um menu principal:

### 🌐 Navegador
- Acessar novas páginas
- Voltar páginas
- Avançar páginas

### 🏥 Hospital
- Gerar senha
- Chamar próximo paciente
- Mostrar histórico de atendimentos

---

## 📥 Entrada e Saída de Dados

- A entrada de dados é realizada pelo teclado.
- A saída de informações é exibida diretamente no console.
